Installation Instructions:

Install:
node.js - https://nodejs.org/en/download/ (we used v4.3.0 for windows)
Inside the SecretWebApp folder (main folder) - run commandline "npm install"

Run:
Inside the SecretWebApp folder (main folder) - run commandline "node index.js"

Connect to: 127.0.0.1:3000 on your webpage
